Implementing Modern Checkbox

  1. Download the checkbox.css file from above and save it to your project folder
  2. Add the checkbox.css file path (if the html and checkbox.css have same path, one can copy this exact code) : 
    <head>
    <link rel="stylesheet" type="text/css" href="checkbox.css">
</head>
  3. Add this where ever you want to put the checkbox : 
    <label class="checkbox">
    <input type="checkbox">
    <span class="slider"></span>
</label>
  4. At the end, your code should look like this : 
    <html>
    <head>
        <link rel="stylesheet" type="text/css" href="checkbox.css">
    </head>
    <body>
        . . . 
        <label class="checkbox">
            <input type="checkbox">
            <span class="slider"></span>
        </label>
        . . . 
    </body>
</html>


Changing Checkbox Properties

  1. Open checkbox.css
  2. Under :root, you can change the :
    --checkbox-height : must, as this calculates & sets all the sizing of the checkbox
    --bg-unchecked : sets the checkbox's color when unchecked
    --bg-checked : sets the checkbox's color when checked
    --bg-slider : sets the rounded slider's color
    by default : 
    :root {
    --checkbox-height: 600px;
    --bg-unchecked: #ccc;
    --bg-checked: #2196f3;
    --bg-slider: #fff;
}
